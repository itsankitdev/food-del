## ✨ About

A full stack food delivery web application where users can browse food items, add them to cart, sign up/sign in, and place orders with payment integration.

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| ⚛️ React | Frontend Framework |
| 🟢 Node.js | Backend Server |
| 🍃 MongoDB | Database |
| 🔐 JWT | Authentication |
| 💳 Payment Gateway | Order Payment |
| 📦 Vercel | Deployment |

---

## 🎯 Features

- ✅ User Authentication (Sign In & Sign Up)
- ✅ JWT based secure login system
- ✅ Browse food items by category
- ✅ Add to Cart functionality
- ✅ Payment integration
- ✅ Order placement system
- ✅ Fully Responsive design

## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/itsankitdev/food-del.git

# 2. Go into the folder
cd food-del

# 3. Install frontend dependencies
cd frontend
npm install

# 4. Install backend dependencies
cd ../backend
npm install

# 5. Create .env file in backend
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret

# 6. Run backend
npm run server

# 7. Run frontend (new terminal)
cd frontend
npm run dev
```
